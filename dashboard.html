Instructions


  • Registration for each event is to be done separately.

  • For registering for an event, the team leader must follow the following steps:
    1. Select the event to register for.
    2. Select the number of team members (including team leader).
    3. Enter the email and bitotsav Id of each team member.
    4. Each team member must be registered individually on the website, i.e each member must have their own bitotsav Id.
    5. None of the team member must be registered for that event.
    6. After successful registration for the event, the team leader must mail documents, photos etc. to events@bitotsav.in as mentioned in the rules and regulations of the corresponding event.

  • Bitotsav Championship
    1. Bitotsav championship is awarded to the team whose members win most number of events.
    2. There is separate registration for Bitotsav championship.
    3. Each team must contain exactly 8 members.
    4. Each member can belong to exactly one team.
    5. The team for Bitotsav championship is different from the team registering for the event.
    6. Lets say a team X registers for an event. If X secures a position in an event and all the members of team X are also registered under the same team Y for Bitotsav championship, then team Y would automatically get points for that event as soon as the result is declared.
